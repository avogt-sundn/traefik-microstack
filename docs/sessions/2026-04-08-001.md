## Hard-won insights captured from existing agent memories

1. **Quarkus-partner routing is permanently broken for /api/partner/search** — Spring-partner must run at priority 1001 to win. This is a design workaround, not a fix. See ADR-0002.

2. **Angular Docker builds require path rewriting** — `federation.config.js`, `tsconfig.app.json extends`, and `angular.json root` paths all differ between local dev and Docker context. See `angular-build-fixer` memory `docker-workspace-layout.md`.

3. **Angular remote base-href must be set explicitly** — `--base-href /partner/` etc. or Angular Router will redirect to the shell instead of rendering the remote.

4. **`public-api.ts` stale exports cause production build failures** — Domain apps import shared items directly by path, so stale entries in `public-api.ts` only surface during Docker builds (not local dev). See `angular-build-fixer` memory.

5. **docker-socket-proxy only exposes CONTAINERS:1** — If Traefik service discovery breaks, check proxy health and network membership before anything else.
