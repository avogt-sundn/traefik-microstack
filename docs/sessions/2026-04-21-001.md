## Hard-won insights

- **OpenAPI generator produces wrong paths for this project**: Generated `localVarPath` values use bare resource names (`/partners/{id}`) but this project routes everything through domain-scoped Traefik paths (`/partner-edit/spring/{id}`). Any newly generated API client needs manual path correction before use.
- **`savePartner` method signature change is a breaking interface change**: The `PartnerGatewayServiceInterface` must be updated in lockstep with the implementation — TypeScript will catch mismatches only if the interface is accurate.
- **Unused injections after call-site removal cause TS strict warnings**: Each removed method call required removing the corresponding `inject(PartnerGatewayService)` and its import to keep the build clean.
- **`make rebuild SERVICE=partner-edit` skips unchanged images**: The build-if-changed script hashes git-tracked content. During active development, use `docker compose build partner-edit` directly to force a rebuild.
