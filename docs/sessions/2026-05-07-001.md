## Hard-won insights

- SmolLM2 is English-first; an explicit German system prompt is insufficient to reliably produce German output. The failure mode is intermittent (sometimes works, often code-switches or falls back to English), making it easy to miss during quick manual testing.
- EuroLLM-1.7B is the ideal candidate for European-language browser inference but has no browser-ready exports (ONNX community weights or MLC prebuilts) as of 2026-05-07. Check `https://huggingface.co/onnx-community` and the MLC model registry before evaluating again.
- Model ID conventions per runtime are non-obvious and must not be guessed:
  - Transformers.js: `onnx-community/<ModelName>` (HuggingFace ONNX community org)
  - WebLLM: `<ModelName>-q4f16_1-MLC` (MLC quantization suffix required)
